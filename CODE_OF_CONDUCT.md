# Code of Conduct

## Expected behavior

- Be specific, constructive, and respectful.
- Provide reproducible evidence for bug reports.
- Keep issue discussions focused on the tool behavior and user workflow.

## Unacceptable behavior

- Harassment, insults, threats, or discriminatory language.
- Publishing private credentials, private assets, or proprietary project files without permission.
- Repeated off-topic comments that prevent maintainers from triaging issues.

## Enforcement

Maintainers may edit, hide, lock, or close issues and pull requests that violate this code. Repeated violations may result in blocking from the project.
